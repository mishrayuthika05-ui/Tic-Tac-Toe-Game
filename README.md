# Tic Tac Toe

This is a simple Tic Tac Toe game created using HTML, CSS, and JavaScript. The project was built to practice front-end web development concepts such as DOM manipulation, event handling, game logic, and responsive design.

## Features

* Two-player game (X and O)
* Detects the winner automatically
* Draw detection
* Restart game option
* Simple and responsive user interface

## Technologies Used

* HTML
* CSS
* JavaScript


## Getting Started

1. Download or clone this repository.
2. Open the project folder.
3. Run the `index.html` file in your browser.

No additional software or libraries are required.

## How to Play

* Player X starts the game.
* Players take turns selecting an empty box.
* The first player to match three symbols in a row, column, or diagonal wins.
* If all boxes are filled and no player wins, the game ends in a draw.
